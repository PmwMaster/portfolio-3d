# 🧊 Portfolio 3D - Cristiano

Um website de portfólio single-page moderno e interativo, focado em alta performance e excelência visual.

## ✨ Principais Funcionalidades

- **Cubo Mágico 3D:** Renderizado nativamente na página usando React Three Fiber, com animação autônoma de embaralhamento e interatividade completa com o rato (OrbitControls e iluminação dinâmica).
- **Design Premium (Wow Factor):** Dark Mode profundo realçado por _Glassmorphism_ nas secções de Contacto e Skills. Contém ainda um fundo espacial de partículas 3D.
- **Micro-interações Físicas:** Cursor customizado (glowing) que reage a links e cartões de projeto/tecnologia simulando gravidade tridimensional (efeito Tilt Reactivo).
- **Performance:** Renderização sem travagens utilizando framer-motion para animações de scroll e Vite para construção rápida.

## 🛠️ Stack Tecnológica

- **React 19**
- **Three.js** (Via `@react-three/fiber` & `@react-three/drei`)
- **Tailwind CSS v4**
- **Framer Motion**
- **TypeScript / Vite**

## 🚀 Como Executar Localmente

1. Faça Git Clone do repositório
2. Instale as dependências usando `npm install`
3. Inicie o servidor de testes com `npm run dev`
